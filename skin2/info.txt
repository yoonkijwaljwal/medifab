■ 2. 템플릿 문법 (Cafe24 스킨 전용)
--------------------------------------------------------------------------------
- <!--@layout(/경로)-->     : 공통 레이아웃 상속
- <!--@import(/경로)-->     : 파셜(부분 HTML) 삽입
- <!--@css(/경로)-->         : CSS 로드
- <!--@js(/경로)-->          : JS 로드
- <!--@contents-->           : 본문 영역 삽입 위치
- module="모듈명"             : Cafe24 서버 모듈 바인딩
- {$변수명}                   : 서버에서 주입되는 동적 변수
- data-ez-module="..."       : EZ Editor 모듈 식별자
- <!--#ez="1/N"-->           : EZ Editor 섹션 마커

■ 4. 페이지 렌더링 흐름
--------------------------------------------------------------------------------

[기본 페이지]
  각 페이지 HTML
    → <!--@layout(/layout/basic/layout.html)--> 로 레이아웃 상속
    → layout.html 이 header / sidebar / footer 를 import
    → <!--@contents--> 위치에 페이지 본문 삽입
    → Cafe24 서버가 module="..." 영역에 DB 데이터 바인딩 후 출력

[레이아웃 구조 (layout.html)]
  <head>
    - 공통 CSS (common, layout, ec-base-*)
    - Swiper, Google Fonts
    - basic.js, layout.js, common.js
    - ez/init.js (EZ Editor)
  <body>
    - header.html (상단배너 + 로고 + 메뉴 + 검색 + Google 번역)
    - sidebar.html (모바일 슬라이드 메뉴)
    - #container > #contents > <!--@contents-->
    - footer.html (사업자정보 + 모바일 하단탭 + 우측 퀵메뉴)

■ 12. 개발 시 주의사항
--------------------------------------------------------------------------------
  1. Cafe24 스킨은 서버(Cafe24)에서 렌더링됨 → 로컬 단독 실행 불가
  2. module="..." 과 {$변수} 는 Cafe24 엔진이 처리, 임의 수정 시 오류 가능
  3. layout/basic/css/common.css 는 전체 페이지 영향 → 수정 주의
  4. EZ Editor 마커(<!--#ez="..."-->) 삭제/변경 시 관리자 편집 기능 깨질 수 있음
  5. header/sidebar 카테고리 메뉴가 하드코딩 → 카테고리 변경 시 양쪽 동시 수정 필요
  6. Google Translate 스크립트는 외부 CDN 의존
  7. 자바스크립트 수정 시 Cafe24 기본 모듈 JS와 충돌 주의
